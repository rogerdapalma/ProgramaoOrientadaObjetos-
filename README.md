# ProgramaoOrientadaObjetos

## Os 4 Pilares da POO

- **Encapsulamento:** Esconde os detalhes internos da implementação e protege os dados.
- **Herança:** Permite que novas classes reaproveitem comportamentos de classes existentes.
- **Polimorfismo:** Possibilita que objetos diferentes respondam à mesma mensagem de formas distintas.
- **Abstração:** Foca apenas nas características essenciais de um objeto, ignorando detalhes irrelevantes.
